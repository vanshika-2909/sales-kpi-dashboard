# 📊 Superstore Sales KPI Dashboard

An interactive Power BI dashboard created to analyze and visualize Superstore sales performance using meaningful KPIs, trend analysis, and category-wise insights.

---

## 🚀 Project Overview

This dashboard helps in understanding:
- Overall sales performance
- Profit analysis
- Order trends
- Regional sales distribution
- Product category contribution
- Top-performing products

The goal of this project was to practice:
- Data visualization
- Dashboard design
- KPI reporting
- Business insight generation using Power BI

---

## 🛠️ Tools & Technologies Used

- Microsoft Power BI
- Data Visualization
- KPI Analytics
- Dashboard Design
- Business Intelligence Concepts

---

## 📌 Dashboard Features

### ✅ KPI Cards
Displays:
- Total Sales
- Total Profit
- Total Orders

### 📈 Monthly Sales Trend
Analyzes monthly sales fluctuations and growth patterns.

### 🌍 Sales by Region
Compares sales performance across different regions.

### 🛒 Sales by Category
Shows category-wise contribution using a pie chart.

### 🏆 Top Products Analysis
Highlights top-performing products based on sales.

### 🎛️ Interactive Filters
Users can filter dashboard data dynamically by:
- Region
- Category
